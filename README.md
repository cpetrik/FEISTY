FEISTY
========

The FishErIes Size and functional TYpe model (FEISTY)

- Colleen M. Petrik, Charles A. Stock, Ken H. Andersen, P. Daniël van Denderen, James R. Watson

## Goals
- develop the size-based ecosystem modeling approach of Watson et al. 2014 with functional types and life stages
- apply to the global ocean to understand bottom-up effects
- climate change forecasts
- top-down effects (Ken, Kisei)
- movement (Fernando, Charlie)
- spawning phenology (Colleen, Rebecca Asch)
- regional applications
     - California Current (Colleen, Jerome Fietcher)
     - Alaska (Colleen, CSOPP, UAF)
     - Gulf of Mexico (Colleen)
- add benthic invertebrates
- add mesopelagics (Daniel, Ken)
- compare to size-specific models (Daniel)
